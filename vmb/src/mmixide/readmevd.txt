Introduction:
=============
MMIX is the processor designed and implemented by Donald E. Knuth
You find more information on MMIX on the MMIX home page at

http://mmix.cs.hm.edu

Getting Started:
================
After installing the binaries, you should be able to 
double-click any .mms file to start the interactive
MMIX Visual Debugger. 

You can open the help file using the Help Menu.

You can get examples and instruction on the MMIX home page.
