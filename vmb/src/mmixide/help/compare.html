Name:
CMP $X,$Y,$Z CMP $X,$Y,Z
CMPU $X,$Y,$ZCMPU $X,$Y,Z

Specification:

CMP:
s($X) ← - 1 if s($Y) < s($Z)
0 if s($Y) = s($Z)
+ 1 if s($Y) > s($Z)
CMPU:
s($X) ← - 1 if u($Y) < u($Z)
0 if u($Y) = u($Z)
+ 1 if u($Y) > u($Z)

Timing:

1υ

Description:

Compares two integer numbers. The result can be used to control Branches or Conditional Set instructions.
